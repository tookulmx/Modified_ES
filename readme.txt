== Installation ==

= Minimum Requirements =

* WordPress 4.1 or greater
* PHP version 7.3 or greater
* MySQL version 5.0 or greater
* PHP curl is required
* Some payment gateways require fsockopen support (for IPN access)
* PHP mbstring extension is required.

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/phanes3dp-legacy` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the 3DP Quote Calculator->Settings screen to configure the plugin
